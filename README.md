# 華容放曹：關羽吃寶箱（最短路徑）— 靜態網頁版

這是將你的 Python 版求解器改寫成純前端（HTML/CSS/JS）的版本，適合部署到 GitHub Pages。

## 這版新增
- 求解時鎖定文字編輯框、檔案載入與範例按鈕
- 顯示運行中的反饋（spinner + 訊息）
- 求解完成後自動解鎖

## 使用方式
1) 打開 `index.html`。
2) 載入 `map.txt` 或貼上盤面文字。
3) 按「開始求解（BFS）」。

## 部署到 GitHub Pages
依照 Settings → Pages 設定分支與資料夾即可。
