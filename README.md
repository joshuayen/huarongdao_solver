
# 華容放曹 — 合併高效版（BFS + A*）

- **功能**：
  - 關羽吃寶箱（BFS，保證最短）
  - 曹操脫逃（A*，高效啟發式）
- **效能**：
  - Web Worker 執行求解，主執行緒保持流暢
  - microseconds（μs）級計時：讀檔 / 求解 / 總計
  - 節點統計：展開節點數、訪問狀態數、佇列/開放表峰值
- **合法性**：移動目標區塊必須是空格，禁止重疊（同字母不同棋子也不可穿越）

## 使用
1. 將所有檔案置於同一資料夾（含 `example/map.txt`）
2. 以瀏覽器開啟 `index.html`
3. 載入範例或貼入盤面，再按 **開始求解（BFS）** 或 **曹操脫逃（A*）**
4. 檢視步驟清單、逐步回放與效能/節點統計

## 檔案說明
- `index.html`：頁面骨架，新增 **曹操脫逃（A*）** 按鈕
- `style.css`：樣式
- `app.js`：UI 控制，與 Solver Worker 溝通
- `solver-worker.js`：BFS/A* 求解器（MinHeap，用於 A*）
- `example/map.txt`：範例盤面
